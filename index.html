Student Success Predictor, DMV Exam Pass Rate Model

I built a machine learning pipeline that analyzes student profiles, lesson attendance records, and practice test scores to predict the probability of passing the New York DMV licensing exam. I designed it to help driving schools prioritize coaching resources and improve student pass rates inspired directly by five years of managing a real driving school's data.

  • Cleans and engineers features from raw student records using Pandas, handling missing values and outliers automatically
  • Trains and compares a Logistic Regression and Random Forest classifier with Scikit-learn, achieving 84% prediction accuracy
  • Outputs a fully interactive Jupyter Notebook report with visual breakdowns per student cohort and risk-level recommendations
